git 指令表

　 　 　 　 　 　 　 　 　 　
git config --global [user.name]設定一個全域變數的名字 git config --global 123
git config --global [user.email]這裡是第二個欄位 git config --global 123@gmail.com
git config --global --list顯示所有的設定 git config --global --list
git init (記得先進入你要放的資料夾)初始化本地的儲存庫 git init
git add -A ( git add -A ) ( git add [檔案名] ) 加入檔案進暫存區 git add 123
git commit -m [訊息內容] 加入訊息到此項目(為了記錄更新什麼) git commit -m "更新"
git status 觀看儲存庫目前狀態 git status
git log 顯示修改紀錄 git log
git diff [commit的ID] [commit的ID] 比較兩個紀錄不同之處 --
